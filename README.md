# Paradise Nursery Shopping App

This repository contains a React shopping cart application for Paradise Nursery, an online plant store.

The application includes a landing page with company details, an About Us section, a product listing page, and a shopping cart. Users can view plants by category, add items to the cart, update quantities, remove items, continue shopping, and view the total cart amount.

## Features

- Paradise Nursery landing page with company name and Get Started button
- About Us information about the company
- Plant products organized by category
- Add to Cart functionality
- Shopping cart quantity controls
- Remove/delete cart items
- Total cart amount calculation

## Technologies Used

- React
- Redux Toolkit
- React Redux
- Vite
- CSS
